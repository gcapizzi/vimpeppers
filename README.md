A Vim distribution for my fellow colleagues @ [XPeppers](http://www.xpeppers.com).

To install it use the installer

    curl https://raw.github.com/ivoputzer/vimpeppers/master/install.sh -L -o - | sh

To remove it use the uninstaller

    curl https://raw.github.com/ivoputzer/vimpeppers/master/uninstall.sh -L -o - | sh
